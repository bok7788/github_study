# CLI

## ls(list)

`-` ls 명령어는 현재 작업중인 위체에 존재하는 목록을 보여준다.

`-` 백틱 3개는 코드 블럭을 생성할 수 있다.

``` bash
DEVPIA@DESKTOP-6JIHTHE MINGW64 ~/Desktop/github_study
$ ls
```

## cd

`-` 현재 작업중인 디렉토리(a.k.a. 폴더)를 변경하겠다.

``` bash
DEVPIA@DESKTOP-6JIHTHE MINGW64 ~/Desktop/github_study
$ cd {directory name}
DEVPIA@DESKTOP-6JIHTHE MINGW64 ~/Desktop/github_study
$ cd .. # 내 상위 폴더로 이동
```

## mkdir(make directory)

``` bash
DEVPIA@DESKTOP-6JIHTHE MINGW64 ~/Desktop/github_study
$ mkdir {directory name}
```

## touch

`-` 파일생성

```bash
DEVPIA@DESKTOP-6JIHTHE MINGW64 ~/Desktop/github_study
$ touch {file name}.{확장자}
```
